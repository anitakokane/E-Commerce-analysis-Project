# 🛒 E-Commerce Customer Behavior Analysis

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on an e-commerce dataset to understand customer purchasing behavior, product performance, and sales trends.

The goal of this analysis is to extract meaningful insights that can help businesses improve marketing strategies, understand customer preferences, and increase revenue.

This project was created as a **beginner data analysis project using Python**.

---

# 🎯 Objective

The objective of this project is to analyze e-commerce customer data to identify patterns in customer behavior, product sales, and order trends.

Through data cleaning, feature engineering, and visualization, this project aims to answer key business questions such as:

* Which customers spend the most?
* Which product categories generate the highest revenue?
* Which cities generate the most orders?
* How do delivery times affect customer ratings?

---

# 📊 Dataset Description

The dataset contains information about customer orders from an e-commerce platform, including:

* Customer demographics
* Product categories
* Purchase amount
* Order date
* Delivery time
* Customer ratings

These variables help analyze **customer purchasing patterns and sales performance**.

---

# 🧹 Data Cleaning Steps

Before performing analysis, the dataset was cleaned to ensure accurate results.

The following steps were performed:

* Checked for missing values
* Handled missing values using **mean, median, or mode**
* Verified duplicate records
* Corrected data types
* Prepared the dataset for analysis

---

# ⚙️ Feature Engineering

Some new features were created to improve analysis:

* Extracted **Month and Year** from order dates
* Created grouped variables for better segmentation
* Organized data to analyze trends more effectively

Feature engineering helped reveal deeper patterns in customer behavior.

# 🔍 Key Insights

Some important insights from the analysis include:

* Certain product categories generate significantly higher revenue.
* A few cities contribute the majority of total orders.
* Sales show noticeable variation across different months.
* Delivery time appears to influence customer ratings.
* Customer demographics such as age and gender show differences in purchasing behavior.

These insights can help businesses **optimize marketing strategies and improve customer experience**.
